Projetos

Adivinhe o número secreto

O projeto "Adivinhe o Número Secreto" é um jogo simples onde o usuário deve escolher um número entre 1 e 10 para tentar adivinhar o número secreto gerado aleatoriamente. O jogo permite ao jogador chutar até acertar, e após o acerto, há a opção de começar um novo jogo. É uma forma divertida e interativa de treinar a percepção de números e de lógica.

Quiz Seriados

O projeto Quiz Seriados é um jogo interativo que desafia os usuários a testar seus conhecimentos sobre séries de TV. Com uma série de perguntas, o jogo permite que o jogador escolha as alternativas corretas, e ao final, exiba o número de acertos. O quiz é projetado para ser divertido e educativo, oferecendo aos participantes a oportunidade de se envolver com seu conteúdo de séries favoritas.

Decodificador de texto

O projeto Decodificador de Texto permite que os usuários convertam textos para diferentes tipos de codificação, como cifra de César ou outros formatos. Uma ferramenta que facilita o envio e a decodificação de mensagens secretas de forma simples e prática.
